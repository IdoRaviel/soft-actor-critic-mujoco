# Soft Actor-Critic (SAC) on MuJoCo

A from-scratch PyTorch implementation of the **Soft Actor-Critic** algorithm, trained on
two continuous-control MuJoCo (`gymnasium`, v4) environments: **Reacher** and **Pusher**.

No RL libraries are used — the agent, networks, replay buffer, and training loop are all
implemented directly in PyTorch.

## Status

🚧 Work in progress.

## Environments

- `Reacher-v4`
- `Pusher-v4`
